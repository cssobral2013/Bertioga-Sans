# Bertioga-Sans
Is developed from open-source fonts UnB Pro. This project was born in 2020, after original UnB Pro fonts received 10+ years with no active development,
so this font was created as a continuation of that font's development.

It includes many fixes and remastering (so this font includes 8 weights with matching italics).

Changes to Bertioga Sans (2020 - 2021):
Re-done Bold weight (so it matches Bold Italic in typographic color);
Expanded from 4 to 8 weights, and added italic styles for all weights;
Many other fixes.
